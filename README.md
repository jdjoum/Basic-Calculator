# Basic-Calculator
This code allows users to perform:
* Addition (+)
* Subtraction (-)
* Division (/)
* Multiplication (*)
* Modulus (%)
* Absolute Value (abs)
* Power (^)
* Natural Log (log)
* Exponential Value (exp)

operations on a number or numbers if applicable.
The user can exit the calculator by typing in "x" for any input.
